1. Для первого задания users.json предварительно должен быть пустым

2. для второго задания users.json должен быть заполнен
example:
  [
    {
      "id": 0,
      "name": "Ivan",
      "age": 19
    }
  ]

3. Для третьего задания users.json можно оставить пустым, заполнить и сверить список (postman: post, raw-json)
